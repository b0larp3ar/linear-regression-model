# Linear Regression Project

This is a beginner machine learning project built using Python and Linear Regression.

The project uses customer data to predict yearly amount spent based on features such as:

* Avg. Session Length
* Time spent on App
* Time spent on Website
* Length of Membership

## Libraries Used

* pandas
* numpy
* scikit-learn
* matplotlib
* seaborn

## Concepts Practiced

* Data preprocessing
* Train/test split
* Linear Regression
* Data visualization
* Model prediction

## Files

* `linear_regression.py` → main Python code
* `Customers.csv` → dataset used for training/testing

## What I Learned

* How to load and analyze datasets using pandas
* How to split data into training and testing sets
* How Linear Regression works and how to implement it using scikit learn
* How to visualize relationships between variables using matplotlib and seaborn
* How to make predictions using scikit-learn
